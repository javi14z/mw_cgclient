**Docker commands:**

1. Lift the container with:
```
docker compose up --build -d
```
2. Stop all containers with: 
```
docker stop $(docker ps -q)
```

**Conexion ssh**

Port 2222 of the host mapped to 22 of the container:
```
ssh -p 2222 cognet@localhost
```
